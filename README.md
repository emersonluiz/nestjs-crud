# crud

## Description

sample of crud with nest and mongo

## Installation

```bash
$ npm install

# db
mongo is required (install or run by docker)
```

## Running the app

```bash
run the mongo

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
